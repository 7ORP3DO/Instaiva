## About The Instaiva

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

There are many other Instagram Downloaders are available on the Internet however they are all full of ads so I made this a free and simple Instagram downloader. I made it with the help of [instagram-web-api](https://www.npmjs.com/package/instagram-web-api). Currently it has only option to download profile picture but more options will be added soon. and If you like it, please consider giving this repo a ‚≠ê.

### Upcoming Features

- [x] Profile Pictures Downloader
- [ ] Instagram Reels Downloader
- [ ] Instagram Stories Downloader
